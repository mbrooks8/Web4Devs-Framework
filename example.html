isLogged.php

We need some information about the user.
We need to know if the current user has permissions to edit the websites content. In the if statement you must set the conditions that allow a user to edit the page. This function will be used to ensure that only users with the correct permissions are able to edit content.

<?php
function isLogged(){
    session_start();
    if(isset($_SESSION['username']))//an example using a session variable
    {
        return 1;
    }

?>
